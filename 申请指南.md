## ⭐ Key 1：智谱 AI（用于生成英文参考回答，永久免费）

**为什么选它**：GLM-4-Flash 模型永久免费，无需充值，不扣费

### 步骤

1. 打开 https://open.bigmodel.cn/ ，点击右上角「注册」
2. 用手机号注册并登录
3. 登录后，点击右上角头像 → **「API Keys」**
4. 点击 **「添加新的 API Key」**
5. 复制生成的 Key（格式类似 `xxxxxxxx.xxxxxxxx`）

✅ 拿到了 `ZHIPU_API_KEY`

## ⭐ Key 2：百度翻译（用于英→中翻译，每月免费 5 万字）

**为什么选它**：免费额度大（5万字/月），一场面试约用 2000 字，完全够用

### 步骤

1. 打开 https://fanyi-api.baidu.com/ ，点击「登录/注册」
2. 注册后进入控制台，找到 **「通用翻译 API」**
3. 点击 **「立即使用」** → 选择「通用翻译（标准版）」→ **免费开通**
4. 开通后，在「开发者信息」页面可以看到：
   - **APP ID**（一串数字）
   - **密钥**（一串字母数字）

✅ 拿到了 `BAIDU_TRANSLATE_APPID` 和 `BAIDU_TRANSLATE_KEY`

## 🔧 填入配置

打开 `server/.env` 文件（从 `.env.example` 复制），找到这三行填入：

```bash
# 百度翻译
BAIDU_TRANSLATE_APPID=你的APP_ID
BAIDU_TRANSLATE_KEY=你的密钥

# 智谱AI
ZHIPU_API_KEY=你复制的Key
```

保存，然后启动后端即可！

**Q: 智谱的 GLM-4-Flash 真的永久免费吗？会不会突然扣费？**
A: 是的，智谱官方公告 GLM-4-Flash 永久免费，不消耗任何额度。你可以在控制台看到用量都是 0 费用。

**Q: 百度翻译免费版有什么限制？**

**Q: 如果我只申请了一个 Key 能用吗？**
